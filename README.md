### YTMusicCli | A new way to listen to music.
An open source Youtube Music Client made with electron.


## Features

- Light/dark mode toggle,
- Light and fast,
- Supports nearly all operating systems,
- Its open source!
## Installation
```bash
  npm install
  // or..
  yarn
```
## Starting the app

```bash
  electron .
```
