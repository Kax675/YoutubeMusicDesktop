## Youtube Music Desktop | A New Way To Listen Music On Your Desktop*
An open source Youtube Music Client made with electron.


## Features

- Light/dark mode toggle,
- Light and fast,
- Supports nearly all operating systems,
- Its open source!
## Installation
```bash
  npm install
  // or..
  yarn
```
## Starting the app

```bash
  electron .
```
